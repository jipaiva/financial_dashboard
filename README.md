# Financial App
A personal finance dashboard to track income, expenses, and savings over time.

## Description
The app helps users monitor their financial habits through automatic monthly tracking and data visualization. It provides a clear overview of spending and saving patterns over time.

## Features
- Track monthly income and expenses
- Automatic creation of monthly entries
- Historical data storage and visualization
- Charts for spending vs. savings comparison

## Tech Stack
- Frontend: Vue.js
- Backend: Node.js, Express.js
- Database: SQLite (sqlite3)

## Status
Work in progress: Currently improving backend structure and adding new features.
